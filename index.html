Primer Parcial

Bienvenida maestra al portafolio del primer parcial que documenta mi viaje a través del curso de Introducción a HTML. Este portafolio es una representación de mi aprendizaje y desarrollo en el manejo de HTML, el lenguaje de marcado fundamental para la creación de páginas web. A lo largo de este curso, he adquirido habilidades clave que son esenciales para cualquier aspirante a desarrollador web, y este portafolio sirve como una ventana a mi progreso y entendimiento del tema.

Si bien no es el portafolio que entrege en el primer parcial dado que lo reemplace por el del segndo, esto solo refleja que en primer parcial solo vimos lo más básico de HTML

Segundo Parcial

Continuando con el segundo parcial, aqui se verá una compilación de mi trayectoria y logros en el curso especializado en CSS (Cascading Style Sheets) y el diseño de estilos para páginas web. Este curso ha sido fundamental para profundizar en cómo CSS mejora la presentación visual de las páginas web, permitiéndome explorar y dominar las técnicas de diseño web moderno. A lo largo de este portafolio, presentaré proyectos y ejercicios que reflejan mi habilidad para transformar estructuras HTML básicas en experiencias web ricas y visualmente atractivas.

Portafolio Segundo Parcial

Tercer Parcial

Para terminar, tenemos el portafolio del tercer parcial, el cual documenta mi experiencia y progreso en el aprendizaje del lenguaje de programación JavaScript, con un enfoque particular en su aplicación para el desarrollo de gráficos interactivos y la generación de archivos PDF. Este curso ha sido esencial para entender la capacidad de JavaScript no solo como una herramienta para añadir interactividad a las páginas web, sino también como un medio poderoso para manejar datos complejos y generar contenido dinámico.
